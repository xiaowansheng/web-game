# 角色扮演游戏 (RPG)

一个经典的角色扮演游戏，玩家控制角色进行冒险、战斗和完成任务，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
角色扮演游戏是一种经典的冒险游戏，玩家控制一个或多个角色进行冒险，与敌人战斗，完成任务，提升角色能力，探索游戏世界。

## 操作说明
- **移动角色**: 使用方向键或WASD键控制角色移动
- **与NPC对话**: 靠近NPC并按空格键
- **打开菜单**: 按ESC键打开游戏菜单
- **战斗**: 在战斗界面使用鼠标点击技能或物品进行战斗
- **新游戏**: 点击"新游戏"按钮开始一局新游戏
- **保存/加载**: 在游戏菜单中进行保存和加载操作

## 游戏规则
1. 玩家控制角色在游戏世界中移动
2. 与NPC对话可以接受任务
3. 完成任务可以获得经验值和奖励
4. 与敌人战斗可以获得经验值和物品
5. 角色升级可以提升属性和学习新技能
6. 使用物品可以恢复生命值和魔法值
7. 游戏目标是完成主线任务，探索整个游戏世界

## 功能特性
- 角色创建和自定义
- 实时战斗系统
- 任务系统
- 角色升级和技能系统
- 物品和装备系统
- 游戏世界探索
- NPC互动
- 保存和加载游戏
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 创建角色，使用方向键或WASD控制角色开始冒险！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 战斗系统设计
- 任务系统实现
- 角色属性和技能管理
- 游戏世界地图系统

## 开发说明
游戏使用模块化设计，将战斗系统、任务系统、角色系统等功能分离，便于扩展和维护。游戏世界采用网格地图系统，支持大型游戏世界的探索。
