# 太空侵略者游戏 (Space Invaders)

一个经典的太空射击游戏，玩家控制飞船射击外星侵略者，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
太空侵略者是一款经典的街机射击游戏，玩家控制一艘飞船在屏幕底部移动，射击从屏幕顶部入侵的外星舰队。游戏目标是消灭所有外星侵略者，同时避免被敌人的攻击击中。

## 操作说明
- **移动飞船**: 使用左右方向键或A/D键控制飞船左右移动
- **射击**: 使用空格键或上方向键/W键发射子弹
- **新游戏**: 刷新页面或在游戏结束后点击"新游戏"按钮

## 游戏规则
1. 玩家控制飞船在屏幕底部移动
2. 外星侵略者从屏幕顶部入侵，左右移动并逐渐向下推进
3. 玩家需要射击外星侵略者
4. 击中每个外星侵略者可以获得分数
5. 外星侵略者会随机发射子弹
6. 如果玩家被敌人的子弹击中，会失去一条生命
7. 如果外星侵略者到达屏幕底部，游戏结束
8. 消灭所有外星侵略者即可通关，进入下一关

## 功能特性
- 实时分数显示
- 生命值系统
- 多级难度递增
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用方向键或WASD控制飞船，按空格键射击开始游戏！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 碰撞检测算法
- 敌人AI移动模式
- 子弹物理运动

## 开发说明
游戏使用JavaScript实现了敌人的移动模式和射击逻辑，通过Canvas绘制游戏元素并实现流畅的动画效果。游戏难度会随着关卡的增加而递增，提供更具挑战性的游戏体验。
