# 迷宫游戏 (Maze Game)

一个经典的迷宫探索游戏，玩家需要控制角色走出迷宫，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
迷宫游戏是一种经典的益智游戏，玩家需要控制角色在复杂的迷宫中找到出口。游戏目标是在最短的时间内走出迷宫。

## 操作说明
- **移动角色**: 使用方向键或WASD键控制角色移动
- **新游戏**: 点击"新游戏"按钮生成新的迷宫
- **重置游戏**: 点击"重置"按钮重新开始当前迷宫

## 游戏规则
1. 游戏在随机生成的迷宫中进行
2. 玩家控制角色从起点出发
3. 角色只能在迷宫的通道中移动，不能穿过墙壁
4. 找到迷宫的出口即可通关
5. 游戏记录通关时间和移动步数

## 功能特性
- 随机生成不同难度的迷宫
- 实时时间和步数统计
- 自动生成的起点和终点
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用方向键或WASD控制角色开始探索迷宫！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 算法: 迷宫生成算法（如深度优先搜索或递归回溯）
- 碰撞检测

## 开发说明
游戏使用深度优先搜索算法随机生成迷宫，确保每个迷宫都有唯一的解决方案。迷宫的大小和复杂度可以根据难度级别进行调整。
