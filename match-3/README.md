# 三消游戏 (Match 3)

一个经典的三消益智游戏，玩家需要交换相邻的元素来形成三个或更多相同的元素连线，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
三消游戏是一种流行的益智游戏，玩家通过交换相邻的元素来形成三个或更多相同元素的水平或垂直连线，消除它们并获得分数。游戏目标是在规定时间或步数内获得尽可能高的分数。

## 操作说明
- **交换元素**: 点击并拖动相邻的元素进行交换
- **新游戏**: 刷新页面或点击"新游戏"按钮开始一局新游戏

## 游戏规则
1. 游戏在网格上进行，网格中分布着各种颜色或形状的元素
2. 玩家可以交换相邻的两个元素
3. 如果交换后形成三个或更多相同元素的水平或垂直连线，则这些元素会被消除
4. 消除元素后，上方的元素会下落填补空缺，新的元素会从顶部出现
5. 连续消除可以形成连击，获得额外分数
6. 游戏有时间限制或步数限制，在规定条件下获得尽可能高的分数

## 功能特性
- 实时分数显示
- 连击系统
- 时间或步数限制
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 点击并拖动元素开始消除！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 算法: 匹配检测和元素下落逻辑
- 随机生成元素

## 开发说明
游戏使用HTML5 Canvas进行渲染，确保流畅的动画效果和良好的用户体验。游戏逻辑采用模块化设计，便于扩展新的游戏模式和功能。
