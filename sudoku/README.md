# 数独游戏 (Sudoku)

一个经典的数独益智游戏，支持不同难度级别，使用HTML、CSS和JavaScript开发。

## 游戏介绍
数独是一种经典的数字填充益智游戏，玩家需要在9x9的方格中填入数字1-9，使得每行、每列和每个3x3的宫格内都包含1-9的数字，且不重复。

## 操作说明
- **选择难度**: 使用下拉菜单选择简单、中等或困难难度
- **新游戏**: 点击"新游戏"按钮开始一局新游戏
- **输入数字**: 点击空白格子，然后使用键盘或数字面板输入数字
- **检查答案**: 点击"检查答案"按钮检查当前解答是否正确
- **提示**: 点击"提示"按钮获取一个正确数字的提示
- **自动求解**: 点击"自动求解"按钮让游戏自动完成数独

## 游戏规则
1. 每行必须包含数字1-9，且不重复
2. 每列必须包含数字1-9，且不重复
3. 每个3x3的宫格必须包含数字1-9，且不重复
4. 初始时，棋盘上已经有一些预设数字
5. 玩家需要填充剩余的空白格子

## 功能特性
- 支持三种难度级别（简单、中等、困难）
- 实时计时器和步数统计
- 答案检查功能
- 提示功能
- 自动求解功能
- 游戏状态提示
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 选择难度，点击"新游戏"开始游戏！

## 技术实现
- HTML: 游戏界面结构
- CSS: 样式和响应式设计
- JavaScript: 游戏逻辑和交互
- 算法: 数独生成和求解算法
- 数组操作: 棋盘状态管理
