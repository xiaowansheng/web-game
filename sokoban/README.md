# 推箱子游戏 (Sokoban)

一个经典的推箱子益智游戏，包含多个关卡，使用HTML、CSS和JavaScript开发。

## 游戏介绍
推箱子是一种经典的益智游戏，玩家需要控制角色将箱子推到指定的目标位置。游戏中需要运用策略，合理规划移动路径，避免箱子被推到无法移动的位置。

## 操作说明
- **移动角色**: 使用 WASD 键或 方向键 控制角色移动
- **新游戏**: 点击"新游戏"按钮开始一局新游戏
- **重置关卡**: 点击"重置关卡"按钮重新开始当前关卡
- **提示**: 点击"提示"按钮获取游戏提示
- **下一关**: 点击"下一关"按钮进入下一个关卡
- **选择关卡**: 点击关卡选择区域的关卡编号直接进入对应关卡

## 游戏规则
1. 玩家可以向上、下、左、右四个方向移动
2. 玩家可以推动箱子，但只能推，不能拉
3. 箱子只能被推到相邻的空白格子或目标位置
4. 玩家不能穿过墙壁或箱子
5. 当所有箱子都被推到目标位置时，关卡通关
6. 游戏记录移动步数和时间

## 功能特性
- 多个精心设计的关卡
- 实时步数和时间统计
- 关卡选择功能
- 提示功能
- 游戏状态提示
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用方向键或WASD开始游戏！

## 技术实现
- HTML: 游戏界面结构
- CSS: 样式和响应式设计
- JavaScript: 游戏逻辑和交互
- 数组操作: 游戏地图和状态管理
- 算法: 碰撞检测和关卡逻辑
