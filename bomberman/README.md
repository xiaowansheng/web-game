# 炸弹人游戏 (Bomberman)

一个经典的炸弹人游戏，玩家放置炸弹来消灭敌人和破坏障碍物，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
炸弹人是一款经典的动作冒险游戏，玩家控制角色在迷宫中放置炸弹，消灭敌人并收集道具，目标是到达每关的出口。

## 操作说明
- **移动角色**: 使用方向键或WASD键控制角色移动
- **放置炸弹**: 使用空格键放置炸弹
- **新游戏**: 刷新页面或等待游戏结束后点击"新游戏"按钮

## 游戏规则
1. 玩家可以在迷宫中自由移动
2. 玩家可以放置炸弹，炸弹会在几秒后爆炸
3. 炸弹爆炸会破坏可破坏的障碍物
4. 炸弹爆炸会消灭敌人和玩家自己
5. 破坏障碍物可能会掉落道具
6. 收集道具可以增强玩家的能力（如增加炸弹数量、爆炸范围等）
7. 消灭所有敌人并到达出口即可通关

## 功能特性
- 多个精心设计的关卡
- 多种敌人类型
- 道具系统
- 实时分数显示
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用方向键或WASD和空格键开始游戏！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 碰撞检测算法
- 爆炸范围计算
- 道具系统
