# 贪吃蛇游戏 (Snake)

一个经典的贪吃蛇游戏，玩家控制蛇来吃食物，使蛇变得更长，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
贪吃蛇是一款经典的休闲游戏，玩家控制一条蛇在屏幕上移动，吃食物来增长身体。游戏目标是让蛇变得尽可能长，同时避免撞到墙壁或自己的身体。

## 操作说明
- **控制方向**: 使用方向键或WASD键控制蛇的移动方向
- **暂停游戏**: 按空格键暂停/继续游戏
- **新游戏**: 刷新页面或在游戏结束后点击"新游戏"按钮

## 游戏规则
1. 游戏开始时，蛇在屏幕中心，长度为3
2. 蛇会自动向前移动
3. 玩家可以改变蛇的移动方向
4. 吃到食物后，蛇的身体会增长一节，同时得分增加
5. 食物被吃后，会在屏幕上随机生成新的食物
6. 如果蛇撞到墙壁或自己的身体，游戏结束
7. 得分与蛇的长度成正比

## 功能特性
- 实时分数显示
- 蛇身长度随得分增长
- 随机生成食物位置
- 游戏暂停功能
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用方向键或WASD控制蛇开始游戏！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 蛇身运动和碰撞检测
- 食物随机生成

## 开发说明
游戏使用JavaScript实现了蛇的运动逻辑和碰撞检测，通过改变蛇头的位置并移动蛇身来模拟蛇的移动效果。游戏采用模块化设计，便于扩展新的功能和游戏模式。
