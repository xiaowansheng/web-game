# Frogger Game

经典的青蛙过河游戏实现，使用HTML5 Canvas和JavaScript开发。

## 游戏说明

控制青蛙从底部的草地出发，穿过繁忙的道路和危险的河流，到达顶部的安全区域。每成功到达一次顶部，玩家将获得分数并进入下一关，难度会逐渐增加。

## 游戏规则

- 使用方向键控制青蛙移动
- 躲避道路上的汽车
- 在河流中，必须跳到漂浮的原木上才能安全通过
- 掉进水里或被汽车撞到会失去一条生命
- 初始有3条生命
- 成功到达顶部获得100分×当前关卡数
- 随着关卡提升，汽车和原木的速度会增加

## 游戏特色

- 响应式控制
- 渐进式难度
- 实时计分系统
- 生命值系统
- 暂停/继续功能
- 游戏结束画面

## 如何运行

1. 克隆或下载本项目
2. 进入frogger目录
3. 可以使用以下方式运行：
   - 直接在浏览器中打开index.html文件
   - 或者使用本地服务器：`npm run dev` 或 `python -m http.server 8080`
4. 点击"Start Game"按钮开始游戏

## 控制说明

- ↑ 向上移动
- ↓ 向下移动
- ← 向左移动
- → 向右移动
- Start Game 开始游戏
- Pause 暂停/继续游戏
