# 2048游戏 (2048)

一个经典的数字合并益智游戏，使用HTML、CSS和JavaScript开发。

## 游戏介绍
2048是一款数字合并类益智游戏，玩家通过移动数字方块来合并相同的数字，目标是合并出2048这个数字。

## 操作说明
- **移动方块**: 使用方向键或WASD键移动方块
- **新游戏**: 刷新页面或等待游戏结束后点击"新游戏"按钮

## 游戏规则
1. 游戏在4x4的网格上进行
2. 每次移动时，所有数字方块会向指定方向移动
3. 相同数字的方块在碰撞时会合并成一个更大的数字
4. 每次移动后，随机位置会生成一个2或4的新方块
5. 当网格填满且无法再移动时，游戏结束
6. 合并数字时，会获得相应的分数

## 功能特性
- 实时分数显示
- 最高分数记录
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用方向键或WASD开始游戏！

## 技术实现
- HTML: 游戏界面结构
- CSS: 样式和响应式设计
- JavaScript: 游戏逻辑和动画
- 数组操作: 游戏网格和状态管理
