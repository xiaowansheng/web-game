# 弹球游戏 (Pinball)

一个经典的弹球游戏，玩家控制挡板来弹击小球，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
弹球是一款经典的街机游戏，玩家通过控制底部的挡板来弹击小球，使小球碰撞各种障碍物和目标，从而获得分数。游戏目标是获得尽可能高的分数。

## 操作说明
- **控制挡板**: 使用鼠标移动或左右方向键/WASD键控制挡板移动
- **发射小球**: 点击"发射"按钮或空格键发射小球

## 游戏规则
1. 游戏开始时，小球位于发射器中
2. 玩家可以调整发射力度，然后发射小球
3. 小球在屏幕上弹来弹去，碰撞各种障碍物
4. 碰撞特定目标可以获得分数
5. 如果小球落到屏幕底部，玩家失去一条生命
6. 当生命值耗尽时，游戏结束

## 功能特性
- 实时分数显示
- 生命值系统
- 多种障碍物和目标
- 得分动画效果
- 流畅的物理运动
- 响应式设计，适配不同屏幕大小

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用鼠标或键盘控制挡板，点击发射按钮开始游戏！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和物理引擎
- CSS: 样式和响应式设计
- 碰撞检测算法
- 物理运动模拟

## 开发说明
游戏使用JavaScript实现了简单的物理引擎，模拟小球的运动和碰撞效果。挡板的控制支持鼠标和键盘两种方式，提高了游戏的可玩性。
