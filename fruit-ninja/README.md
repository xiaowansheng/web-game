# 水果忍者游戏 (Fruit Ninja)

一个经典的水果忍者游戏，玩家通过滑动屏幕来切水果，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
水果忍者是一款流行的休闲游戏，屏幕上会不断出现各种水果，玩家需要通过滑动屏幕来切割水果，同时避开炸弹。

## 操作说明
- **切割水果**: 使用鼠标滑动或触摸屏幕来切割水果
- **新游戏**: 刷新页面或等待游戏结束后点击"新游戏"按钮

## 游戏规则
1. 屏幕上会随机出现各种水果
2. 玩家需要通过滑动屏幕来切割水果
3. 切割每个水果可以获得分数
4. 同时切割多个水果可以获得额外的连击分数
5. 避开炸弹，切到炸弹会减少生命值
6. 当生命值耗尽时，游戏结束
7. 游戏有时间限制，在规定时间内获得尽可能高的分数

## 功能特性
- 实时分数显示
- 连击系统
- 炸弹躲避机制
- 时间限制
- 生命值系统
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用鼠标滑动或触摸屏幕开始切水果！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 鼠标/触摸事件处理
- 碰撞检测算法
- 随机生成水果和炸弹
