# 记忆翻牌游戏 (Memory Card)

一个经典的记忆翻牌游戏，玩家需要记住卡片的位置并找出配对，使用HTML、CSS和JavaScript开发。

## 游戏介绍
记忆翻牌是一款经典的益智游戏，玩家需要翻转卡片来找到配对的图案。游戏目标是用最少的步数和最短的时间找出所有配对的卡片。

## 操作说明
- **翻转卡片**: 点击卡片进行翻转
- **新游戏**: 点击"新游戏"按钮开始一局新游戏

## 游戏规则
1. 游戏开始时，所有卡片背面朝上
2. 玩家每次可以翻转两张卡片
3. 如果两张卡片的图案相同，则它们会保持正面朝上
4. 如果两张卡片的图案不同，则它们会翻回背面
5. 游戏目标是找出所有配对的卡片
6. 游戏记录翻转次数和用时

## 功能特性
- 实时步数和时间统计
- 多种难度级别（不同数量的卡片）
- 流畅的翻转动画效果
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 点击卡片开始游戏！

## 技术实现
- HTML: 游戏界面结构
- CSS: 样式和响应式设计
- JavaScript: 游戏逻辑和动画
- 随机洗牌算法
- 事件处理和状态管理

## 开发说明
游戏使用CSS 3D变换实现卡片的翻转动画效果，增强游戏的视觉体验。卡片图案使用Unicode符号或CSS绘制，确保游戏的轻量级和可移植性。
