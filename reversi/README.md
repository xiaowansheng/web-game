# 黑白棋游戏 (Reversi/Othello)

一个经典的黑白棋策略游戏，使用HTML、CSS和JavaScript开发。

## 游戏介绍
黑白棋（也称为奥赛罗）是一种经典的策略棋盘游戏，两名玩家分别使用黑棋和白棋进行对战。游戏目标是在游戏结束时，棋盘上自己颜色的棋子数量超过对手。

## 操作说明
- **放置棋子**: 点击棋盘上的空白格子放置棋子
- **新游戏**: 点击"新游戏"按钮开始一局新游戏
- **撤销**: 点击"撤销"按钮撤销上一步操作
- **跳过**: 当无法落子时，点击"跳过"按钮跳过当前回合

## 游戏规则
1. 游戏开始时，棋盘中心放置2个黑棋和2个白棋
2. 黑棋先下，黑白双方交替落子
3. 只能在能翻转对方棋子的位置落子
4. 翻转规则：落子后，对方的棋子被己方棋子在横向、纵向或对角线上包围
5. 无法落子时可以跳过回合
6. 当双方都无法落子时，游戏结束
7. 棋盘上棋子数量多的一方获胜

## 功能特性
- 实时分数显示
- 当前玩家回合提示
- 游戏状态提示
- 撤销功能
- 跳过回合功能
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 开始游戏！

## 技术实现
- HTML: 游戏界面结构
- CSS: 样式和响应式设计
- JavaScript: 游戏逻辑和交互
- 数组操作: 棋盘状态管理
- 算法: 合法落子检测和棋子翻转逻辑
