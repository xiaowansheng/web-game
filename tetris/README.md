# 俄罗斯方块游戏 (Tetris)

一个经典的俄罗斯方块游戏，玩家控制下落的方块，将它们排列成完整的行，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
俄罗斯方块是一款经典的休闲益智游戏，玩家控制各种形状的方块从屏幕顶部下落，通过旋转和移动方块，将它们排列成完整的行。游戏目标是尽可能长时间地生存并获得高分。

## 操作说明
- **移动方块**: 使用左右方向键或A/D键左右移动方块
- **旋转方块**: 使用上方向键或W键旋转方块
- **加速下落**: 使用下方向键或S键加速方块下落
- **直接落地**: 使用空格键让方块直接落到底部
- **暂停/继续**: 按P键或点击"暂停"按钮
- **新游戏**: 点击"新游戏"按钮开始一局新游戏

## 游戏规则
1. 游戏在10x20的网格上进行
2. 不同形状的方块从屏幕顶部随机下落
3. 玩家可以左右移动、旋转方块，或加速其下落
4. 当方块落到屏幕底部或碰到其他方块时停止移动
5. 如果一行被方块填满，该行会消失，玩家获得分数
6. 消除多行可以获得额外的连击分数
7. 当方块堆到屏幕顶部时，游戏结束

## 功能特性
- 实时分数显示
- 消行统计
- 下一个方块预览
- 游戏等级系统（随分数增加难度）
- 暂停功能
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用方向键或WASD控制方块开始游戏！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 方块形状定义和旋转算法
- 碰撞检测
- 消行逻辑和分数计算

## 开发说明
游戏使用JavaScript实现了方块的旋转算法和碰撞检测，确保方块在旋转和移动时不会超出边界或与其他方块重叠。游戏等级系统会随着玩家分数的增加而提高方块的下落速度，增加游戏的挑战性。
