# 打地鼠游戏 (Whack a Mole)

一个经典的打地鼠休闲游戏，玩家通过点击从地洞中冒出的地鼠来得分，使用HTML、CSS和JavaScript开发。

## 游戏介绍
打地鼠是一款经典的休闲游戏，屏幕上会有多个地洞，地鼠会随机从这些地洞中冒出。玩家需要在规定时间内尽可能多地点击地鼠，获得高分。

## 操作说明
- **击打地鼠**: 点击从地洞中冒出的地鼠
- **选择难度**: 使用下拉菜单选择简单、中等或困难难度
- **开始游戏**: 点击"开始游戏"按钮开始一局新游戏

## 游戏规则
1. 游戏在包含多个地洞的面板上进行
2. 地鼠会随机从地洞中冒出，停留一段时间后缩回
3. 玩家需要点击冒出的地鼠来得分
4. 不同难度级别对应不同的游戏时间和地鼠停留时间
5. 游戏结束后，显示最终得分
6. 得分越高，游戏等级越高

## 功能特性
- 支持三种难度级别（简单、中等、困难）
- 实时分数显示
- 游戏倒计时
- 地鼠随机出现和消失
- 击中地鼠的动画效果
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 选择难度，点击"开始游戏"按钮，然后点击冒出的地鼠开始游戏！

## 技术实现
- HTML: 游戏界面结构
- CSS: 样式和响应式设计
- JavaScript: 游戏逻辑和动画
- 随机数生成（地鼠出现位置和时间）
- 点击事件处理
- 计时器功能

## 开发说明
游戏使用JavaScript实现了地鼠的随机出现和消失逻辑，通过CSS动画实现地鼠冒出和被击中的效果。游戏难度系统根据选择的难度调整地鼠停留时间和游戏总时间，提供不同程度的挑战性。
