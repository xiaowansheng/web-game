# 坦克大战游戏 (Tank Battle)

一个经典的坦克对战游戏，支持双人对战，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
坦克大战是一款经典的双人对战游戏，玩家控制坦克在地图上移动，射击敌人的坦克和基地。游戏目标是摧毁敌人的基地，同时保护自己的基地。

## 操作说明
### 玩家1控制
- **移动**: 使用WASD键控制坦克移动
- **射击**: 使用空格键发射子弹

### 玩家2控制
- **移动**: 使用方向键控制坦克移动
- **射击**: 使用Enter键发射子弹

### 其他操作
- **新游戏**: 点击"新游戏"按钮开始一局新游戏
- **暂停/继续**: 按P键或点击"暂停"按钮

## 游戏规则
1. 游戏在包含障碍物的地图上进行
2. 玩家控制坦克在地图上移动和射击
3. 击中敌人的坦克可以摧毁它们并获得分数
4. 击中敌人的基地可以获胜
5. 被敌人的子弹击中会摧毁自己的坦克
6. 障碍物可以阻挡坦克和子弹的移动
7. 有些障碍物可以被摧毁

## 功能特性
- 双人对战模式
- 多种障碍物类型（可摧毁和不可摧毁）
- 实时分数显示
- 基地保护机制
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用WASD和方向键控制坦克，按空格键或Enter键射击开始游戏！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 碰撞检测算法
- 地图加载和障碍物管理
- 坦克和子弹物理运动

## 开发说明
游戏使用JavaScript实现了坦克的移动和射击逻辑，以及地图上的碰撞检测。地图设计采用模块化结构，支持多种地图布局和障碍物类型。双人对战模式支持两个玩家同时进行游戏，增加了游戏的趣味性和竞争性。
