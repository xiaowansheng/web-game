# 赛车游戏 (Racing)

一个经典的赛车游戏，玩家控制赛车在赛道上行驶，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
赛车游戏是一款经典的竞速游戏，玩家控制赛车在赛道上行驶，超越其他车辆，目标是在最短的时间内到达终点。

## 操作说明
- **控制赛车**: 使用左右方向键或A/D键控制赛车左右移动
- **加速/减速**: 使用上/下方向键或W/S键控制赛车加速/减速
- **新游戏**: 点击"新游戏"按钮开始一局新游戏

## 游戏规则
1. 游戏在赛道上进行，玩家控制一辆赛车
2. 赛道上有其他车辆和障碍物
3. 玩家需要避开其他车辆和障碍物
4. 尽可能快地到达终点
5. 碰撞其他车辆或障碍物会减少生命值
6. 生命值耗尽时，游戏结束
7. 到达终点的时间越短，得分越高

## 功能特性
- 多个赛道选择
- 实时速度和时间显示
- 生命值系统
- 多种其他车辆和障碍物
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用方向键或WASD控制赛车开始比赛！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 物理运动模拟
- 碰撞检测算法
- 随机生成车辆和障碍物

## 开发说明
游戏使用JavaScript实现了简单的物理引擎，模拟赛车的运动和碰撞效果。赛道设计采用多层背景实现深度感，增强游戏的视觉体验。
