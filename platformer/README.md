# 平台跳跃游戏 (Platformer)

一个经典的2D平台跳跃游戏，玩家控制角色跳跃、奔跑和收集物品，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
平台跳跃游戏是一种经典的动作游戏，玩家控制角色在不同的平台之间跳跃，躲避障碍物，收集物品，目标是到达关卡的终点。

## 操作说明
- **移动角色**: 使用左右方向键或A/D键控制角色左右移动
- **跳跃**: 使用上方向键或W键控制角色跳跃
- **攻击**: 使用空格键控制角色攻击
- **新游戏**: 点击"新游戏"按钮开始一局新游戏
- **重置关卡**: 点击"重置关卡"按钮重新开始当前关卡

## 游戏规则
1. 玩家控制角色在平台上移动和跳跃
2. 收集金币和其他物品可以获得分数
3. 躲避敌人和障碍物，避免受到伤害
4. 到达关卡终点即可通关
5. 角色有生命值限制，受到伤害会减少生命值
6. 生命值耗尽时，游戏结束

## 功能特性
- 多个精心设计的关卡
- 角色跳跃和攻击机制
- 敌人和障碍物
- 金币和道具收集系统
- 生命值和分数统计
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用方向键或WASD控制角色开始游戏！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 物理运动模拟（重力、跳跃）
- 碰撞检测算法
- 关卡设计和加载

## 开发说明
游戏使用JavaScript实现了重力系统和碰撞检测，模拟真实的跳跃和移动效果。关卡设计采用模块化结构，便于扩展新的关卡和功能。
