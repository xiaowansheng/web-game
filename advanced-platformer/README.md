# 高级平台跳跃游戏

一个基于Phaser.js框架开发的复杂平台跳跃游戏，包含多个关卡、敌人AI和Boss战。

## 功能特性

- 🎮 **多关卡设计**：3个普通关卡 + 1个Boss关卡
- 🤖 **智能敌人AI**：敌人会自动移动和攻击
- 👾 **史诗Boss战**：最终Boss具有复杂的攻击模式
- 🎯 **射击系统**：玩家可以发射子弹攻击敌人
- 🔄 **收集系统**：收集物品获得分数和强化
- 💪 **升级系统**：提升玩家能力
- 📊 **完整UI**：显示分数、生命值和关卡信息
- 🎨 **流畅动画**：使用Phaser.js的物理引擎实现流畅的游戏体验

## 技术栈

- **框架**：Phaser.js 3.x
- **语言**：JavaScript
- **包管理**：npm
- **物理引擎**：Phaser内置Arcade Physics

## 安装

1. 克隆或下载项目到本地
2. 进入项目目录
3. 安装依赖

```bash
cd advanced-platformer
npm install
```

## 运行游戏

直接在浏览器中打开 `index.html` 文件即可开始游戏：

```bash
# 使用默认浏览器打开
index.html

# 或使用本地服务器（推荐）
python -m http.server 8000
# 然后在浏览器中访问 http://localhost:8000
```

## 游戏控制

- **方向键**：移动玩家
- **空格键**：跳跃
- **X键**：发射子弹

## 游戏机制

### 生命值
- 玩家初始有3条生命值
- 接触敌人或掉落会失去生命值
- 生命值为0时游戏结束

### 分数系统
- 收集物品：+100分
- 消灭敌人：+500分
- 击败Boss：+1000分

### 强化系统
- 收集特殊物品可以提升玩家攻击力
- 攻击力提升后，子弹对Boss造成的伤害增加

### 关卡设计
1. **关卡1**：基础平台跳跃，熟悉游戏操作
2. **关卡2**：增加敌人数量，提升难度
3. **关卡3**：复杂的平台布局，大量敌人
4. **Boss关卡**：最终Boss战，需要灵活躲避和攻击

## 开发说明

### 添加新关卡
1. 在 `assets/levels/` 目录下创建新的JSON文件
2. 参考现有关卡格式设计平台、敌人和收集品
3. 在 `GameScene.js` 中添加关卡切换逻辑

### 添加新敌人
1. 在 `js/objects/` 目录下创建新的敌人类
2. 继承或参考 `Enemy.js` 实现敌人AI
3. 在关卡数据中添加新敌人配置

### 修改游戏参数
- 重力、速度等物理参数：修改 `js/game.js` 中的 `config.physics`
- 玩家属性：修改 `Player.js` 中的初始化参数
- Boss属性：修改 `Boss.js` 中的初始化参数
