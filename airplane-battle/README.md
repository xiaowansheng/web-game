# 飞机大战游戏 (Airplane Battle)

一个经典的飞机射击游戏，玩家控制飞机射击敌人，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
飞机大战是一款经典的射击游戏，玩家控制一架飞机在屏幕上移动，射击敌人并躲避敌人的攻击。游戏目标是尽可能长时间地生存并获得更高的分数。

## 操作说明
- **移动飞机**: 使用鼠标移动或方向键/WASD键控制飞机移动
- **射击**: 飞机自动射击，或使用空格键手动射击

## 游戏规则
1. 玩家控制一架飞机在屏幕底部移动
2. 敌人从屏幕顶部随机出现并向下移动
3. 玩家需要射击敌人并躲避敌人的攻击
4. 击中敌人可以获得分数
5. 当玩家被敌人击中或敌人到达屏幕底部时，游戏结束
6. 游戏中会随机出现道具，可以增强玩家的火力或恢复生命值

## 功能特性
- 实时分数显示
- 生命值系统
- 多种敌人类型
- 道具系统
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用鼠标或键盘控制飞机开始游戏！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 碰撞检测算法
- 随机生成敌人和道具
