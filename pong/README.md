# 乒乓球游戏 (Pong)

一个经典的双人乒乓球游戏，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
这是一个经典的乒乓球游戏，支持两名玩家同时进行游戏。游戏目标是将球击打过网，让对手无法接住，从而获得分数。

## 操作说明
- **玩家1**: 使用 W 和 S 键上下移动球拍
- **玩家2**: 使用 方向键上下 移动球拍
- **重新开始**: 点击"重新开始"按钮

## 游戏规则
1. 游戏开始时，球从屏幕中心向随机方向发射
2. 当球碰到球拍时，会根据碰撞位置改变反弹角度
3. 当球越过对方底线时，对方得分
4. 得分后，球会重新从中心发射

## 功能特性
- 实时分数显示
- 响应式设计，适配不同屏幕大小
- 流畅的动画效果
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 开始游戏！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
