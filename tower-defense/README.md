# 塔防游戏 (Tower Defense)

一个经典的塔防策略游戏，玩家通过建造防御塔来阻止敌人入侵，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
塔防游戏是一种经典的策略游戏，玩家需要在敌人的进攻路线上建造各种防御塔，来阻止敌人到达终点。游戏目标是在敌人到达终点前消灭所有敌人，保护自己的基地。

## 操作说明
- **建造防御塔**: 点击地图上的空白区域，然后选择要建造的防御塔类型
- **升级防御塔**: 点击已建造的防御塔，然后点击"升级"按钮
- **出售防御塔**: 点击已建造的防御塔，然后点击"出售"按钮
- **开始波次**: 点击"开始下一波"按钮开始新的敌人进攻
- **新游戏**: 点击"新游戏"按钮开始一局新游戏

## 游戏规则
1. 游戏在包含敌人进攻路线的地图上进行
2. 玩家可以使用初始金币建造防御塔
3. 敌人会按照设定的路线进攻
4. 防御塔会自动攻击范围内的敌人
5. 消灭敌人可以获得金币
6. 金币可以用来建造新的防御塔或升级现有防御塔
7. 如果敌人到达终点，玩家会失去生命值
8. 生命值耗尽时，游戏结束
9. 成功抵御所有波次的敌人即可通关

## 功能特性
- 多种类型的防御塔（不同的攻击范围、伤害和速度）
- 多种类型的敌人（不同的生命值、速度和奖励）
- 金币系统和防御塔升级机制
- 波次进攻系统
- 生命值系统
- 实时游戏状态显示
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 点击地图空白区域建造防御塔，点击"开始下一波"按钮开始游戏！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 防御塔和敌人的AI逻辑
- 路径查找算法
- 碰撞检测和攻击范围计算

## 开发说明
游戏使用JavaScript实现了防御塔的攻击逻辑和敌人的路径查找，通过Canvas绘制游戏元素并实现流畅的动画效果。防御塔和敌人的类型设计采用模块化结构，便于扩展新的防御塔和敌人类型。
