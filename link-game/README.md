# 连连看游戏 (Link Game)

一个经典的连连看益智游戏，玩家需要连接相同的图案来消除它们，使用HTML、CSS和JavaScript开发。

## 游戏介绍
连连看是一款经典的益智游戏，玩家需要在规定时间内找到并连接相同的图案，目标是消除屏幕上的所有图案。

## 操作说明
- **连接图案**: 点击两个相同的图案，如果它们可以通过不多于两条直线连接，则会被消除
- **新游戏**: 刷新页面或点击"新游戏"按钮开始一局新游戏

## 游戏规则
1. 游戏在网格上进行，网格中分布着各种图案
2. 玩家需要找到两个相同的图案
3. 这两个图案之间的连线不能超过两条直线
4. 消除所有图案后，游戏通关
5. 游戏有时间限制，在规定时间内完成游戏

## 功能特性
- 实时时间显示
- 多种难度级别
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 点击相同的图案开始连接！

## 技术实现
- HTML: 游戏界面结构
- CSS: 样式和响应式设计
- JavaScript: 游戏逻辑和动画
- 算法: 路径查找和连接检测
- 随机生成图案

## 开发说明
游戏使用纯前端技术开发，无需后端支持。图案资源使用Unicode符号或CSS绘制，确保游戏的轻量级和可移植性。
