# 剪刀石头布游戏 (Rock Paper Scissors)

一个经典的剪刀石头布游戏，玩家与电脑对战，使用HTML、CSS和JavaScript开发。

## 游戏介绍
这是一个经典的剪刀石头布游戏，玩家可以选择石头、布或剪刀与电脑进行对战。游戏目标是先赢得5个回合，成为最终赢家。

## 操作说明
- 点击屏幕上的石头、布或剪刀图标进行选择
- 游戏会自动与电脑进行对战并显示结果
- 点击"重新开始"按钮可以重置游戏

## 游戏规则
1. 石头击败剪刀
2. 剪刀击败布
3. 布击败石头
4. 每回合胜利得1分
5. 先获得5分的一方获胜
6. 如果双方选择相同，则为平局，不计分

## 功能特性
- 实时分数显示
- 回合数记录
- 游戏结果动画效果
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 开始游戏！

## 技术实现
- HTML: 游戏界面结构
- CSS: 样式和动画效果
- JavaScript: 游戏逻辑和交互
- 随机数生成: 电脑选择逻辑
