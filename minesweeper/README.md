# 扫雷游戏 (Minesweeper)

一个经典的扫雷益智游戏，玩家需要在布满地雷的网格中找出所有地雷，使用HTML、CSS和JavaScript开发。

## 游戏介绍
扫雷是一款经典的益智游戏，玩家需要在布满地雷的网格中，通过点击格子来找出所有地雷的位置。游戏目标是在不触发任何地雷的情况下，找出所有地雷的位置。

## 操作说明
- **翻开格子**: 左键点击格子翻开它
- **标记地雷**: 右键点击格子标记为地雷
- **选择难度**: 使用下拉菜单选择简单、中等或困难难度
- **新游戏**: 点击"新游戏"按钮开始一局新游戏

## 游戏规则
1. 游戏在网格上进行，网格中随机分布着地雷
2. 左键点击格子可以翻开它
3. 如果翻开的格子是地雷，游戏结束
4. 如果翻开的格子不是地雷，会显示周围8个格子中地雷的数量
5. 如果翻开的格子周围没有地雷，会自动翻开周围的格子
6. 右键点击格子可以标记为地雷
7. 找出所有地雷的位置即可通关

## 功能特性
- 支持三种难度级别（简单、中等、困难）
- 实时剩余地雷数量显示
- 计时器
- 右键标记地雷功能
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小
- 简单直观的操作界面

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 选择难度，点击格子开始扫雷！

## 技术实现
- HTML: 游戏界面结构
- CSS: 样式和响应式设计
- JavaScript: 游戏逻辑和动画
- 算法: 地雷随机分布和数字计算
- 事件处理（左键和右键点击）

## 开发说明
游戏使用纯前端技术开发，无需后端支持。地雷的分布和周围数字的计算使用高效的算法实现，确保游戏的流畅性和可玩性。
