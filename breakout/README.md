# 打砖块游戏 (Breakout)

一个经典的打砖块游戏，玩家控制球拍反弹小球来打破砖块，使用HTML5 Canvas和JavaScript开发。

## 游戏介绍
打砖块是一款经典的街机游戏，玩家控制底部的球拍反弹小球，目标是打破屏幕上方的所有砖块。

## 操作说明
- **移动球拍**: 使用鼠标移动或方向键/WASD键控制球拍移动
- **新游戏**: 刷新页面或等待游戏结束后点击"新游戏"按钮

## 游戏规则
1. 玩家控制底部的球拍移动
2. 小球从球拍出发，向上移动
3. 小球碰到砖块会将其打破并反弹
4. 小球碰到墙壁或顶部会反弹
5. 如果小球落到屏幕底部，玩家失去一条生命
6. 当所有砖块都被打破时，游戏通关
7. 打破不同颜色的砖块可能获得不同的分数

## 功能特性
- 实时分数显示
- 生命值系统
- 多种颜色的砖块
- 流畅的动画效果
- 响应式设计，适配不同屏幕大小

## 如何运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 使用鼠标或键盘控制球拍开始游戏！

## 技术实现
- HTML5 Canvas: 游戏渲染
- JavaScript: 游戏逻辑和动画
- CSS: 样式和响应式设计
- 碰撞检测算法
- 小球物理运动
